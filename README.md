## MyLearning
精读学习内容

###To Myself : ###

	不要纠结开发语言，多学习数据结构和算法、网络、数据库、存储、操作系统方面的知识。
	每个编程语言都有适合自己的领域，兼顾即可，根据需求选择最合适的语言去开发。
	语言是工具，架构是经验，底层是基础是核心。
